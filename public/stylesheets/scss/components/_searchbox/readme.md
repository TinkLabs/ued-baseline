### Search Box
Place to input search or filter keywords, can be used together with list

### Usage
``` 
<div class="searchbox">
    <div class="search-wrapper">
        <input class="search-input" type="text" placeholder="search box">
        <ul class="search-history list">
            <li class="list-row">
                <div class="list-content">
                    <div class="list-text">
                        <span class="list-text-secondary"> search history 1 </span>
                    </div>
                </div>
            </li>
            <li class="list-row">
                <div class="list-content">
                    <div class="list-text">
                        <span class="list-text-secondary"> search history 2 </span>
                    </div>
                </div>
            </li>
        </ul>
    </div>
    <p class="search-helper"> helper text </p>
<div>

```
