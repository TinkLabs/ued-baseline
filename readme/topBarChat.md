### Top Bar (Chat)
Provides user with control and contact info of a chat

### Usage
```html
<div class="top-bar top-bar-chat">
    <div class="top-bar-left">
        <div class="top-bar-icon">
            <i class="icon icon-hardware_keyboard_arrow_left_24px"></i>
        </div>
    </div>
    <div class="top-bar-body">
        <div class="chat-info-strip">
            <img class="avatar" src="./public/images/checker_bg.png" alt="avatar">
            <div class="chat-info">
                <p class="chat-name">Name</p>
                <p class="chat-number">+123 123 456 789</p>
            </div>
        </div>
    </div>
    <div class="top-bar-right">
        <div class="top-bar-actions">
            <i class="icon icon-maps_local_phone_24px"></i>
        </div>
    </div>
</div>
```
